ani-web

A lightweight, local anime streaming web app designed to run smoothly on any hardware.

Quick Start

Install and run ani-web with a single command. After the first run, you can start the app anytime by just typing ani-web in your terminal.

Windows

irm "https://serifpersia.github.io/ani-web/setup.ps1" | iex

Linux & macOS

curl -sSL "https://serifpersia.github.io/ani-web/setup.sh" | bash

Features

  • Lightweight & Fast
  • Simple, Clean UI
  • Local Watchlist
  • MyAnimeList Integration
  • Continue Watching
  • No Account Needed
